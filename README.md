# Ptero Watchdog

A detecting system for miners, tunnels, xmrig, and more with double detecting methods of file checking for file names and process checking for checking suspicious process incase of file detection fail or files deleted but process running.

## Installation Guide:

- Configure ``config.json`` file with directories, file keywords, process keywords that the script will detect.
- run it as root on wings server.
- Get live logs on Discord via webhooks.


## AI Usage:

- AI is used to research on features and methods to detect miners, tunnels, etc and few bug fixes.
